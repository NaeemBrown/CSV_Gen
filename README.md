# Standard Invoice (PDF) to Excel xcls converter                        
**Automating 'Excel Hell' for local trade businesses.**

This Python-based tool (built with `CustomTkinter` and `pdfplumber`) allows small businesses to batch-convert messy supplier PDF invoices into consolidated Excel sheets in seconds.

### Features 
* **Batch Processing:** Drop a whole folder of invoices and process them all at once.
* **Smart Extraction:** Targets tables specifically to avoid "text junk."
* **Zero Installation:** Can be compiled to `.exe` for non-tech users.
